# Projeto Java com Maven

Este projeto utiliza **GitHub Actions** para integração contínua (CI).

## Workflow de CI
- Nome: Java CI with Maven
- Localização: `.github/workflows/maven.yml`
- Função: Compilar e executar testes automaticamente após cada push para a branch `main`.
